arrival_date *
Date (yyyy-mm-dd)

The date on which you will arrive or check-in

departure_date *
Date (yyyy-mm-dd)

The date of departure or check-out.

adults (optional)
Number

The number of guests who are 18 years of age or older. The default value is set to 1.

children_age (optional)
String

The number of children, including infants, who are under 18. Example: Child 1 Age = 8 months Child 2 Age = 1 year Child 3 Age = 17 years Here is what the request parameter would look like: children_age: 0,1,17

room_qty (optional)
Number

The number of rooms that are required. The default value is set to 1.

page_number (optional)
Number

The page number.

price_min (optional)
Number

Minimum Price filter for search.

price_max (optional)
Number

Maximum Price filter for search.

categories_filter (optional)
String

categories_filter can be retrieved from api/v1/hotels/getFilter(Get Filter) endpoint in Hotels collection.

units (optional)
metric
Enum

The measurement of distance in metric or imperial.

Enum
